#### theodolit.py
Contoh kode untuk menghitung data ukur theodolit

Input :
- Global Base Point N *(N)*
- Global Base Point E *(E)*
- Global Base Point Z *(Z)*

- Tinggi Alat *(HI)*
- Benang Tengah *(BT)*
- Benang Atas *(BA)*

Output :
- Jarak miring *(SD)*
- Jarak datar *(HD)*

- Delta N *(dN)*
- Delta E *(dE)*
- Delta Z *(dZ)*

- Global N *(N)*
- Global E *(E)*
- Global Z *(Z)*
